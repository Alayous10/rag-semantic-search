# Semantic Document Search Engine (RAG) 🚀

محرك بحث دلالي ذكي للمستندات يعتمد على تقنية **RAG** (Retrieval-Augmented Generation). 
يسمح هذا النظام للمستخدمين برفع مستندات متعددة الأنواع (PDF, TXT, DOCX) ثم طرح أسئلة باللغة الطبيعية حول محتواها، ليقوم النظام بتحليل المستند وتوليد إجابة دقيقة مستندة حصرياً على النص المرفوع.

## 🛠️ التقنيات المستخدمة (Tech Stack)

### الواجهة الخلفية (Backend)
- **Python 3.12**
- **FastAPI:** لبناء مسارات API سريعة وفعالة.
- **LangChain:** للتحكم في تدفق الذكاء الاصطناعي وتقسيم النصوص (Chunking).
- **ChromaDB:** قاعدة بيانات متجهة (Vector Store) تعمل محلياً وبدون خوادم.
- **Gemini 1.5 Flash & embedding-001:** نماذج الذكاء الاصطناعي الأساسية لمعالجة اللغات الطبيعية.

### الواجهة الأمامية (Frontend)
- **Next.js 15 (App Router)**
- **Tailwind CSS:** لتصميم واجهة مستخدم (UI) مظلمة وجذابة (Dark Cyberpunk).
- **Axios & Lucide React:** لإدارة الطلبات والأيقونات.

## 💻 كيفية التشغيل المحلي (Running Locally)

لتشغيل هذا المشروع على جهازك، نوصي باستخدام **Docker** لتسهيل إعداد البيئة وتجنب مشاكل الاعتماديات.

### المتطلبات المسبقة:
- تثبيت [Docker](https://www.docker.com/products/docker-desktop/) على جهازك.
- الحصول على مفتاح API صالح من جوجل (Gemini API Key).

### خطوات التشغيل:

1. **إعداد مفتاح الـ API:**
   تأكد من إنشاء ملف `.env` داخل المجلد `backend/` وإضافة مفتاحك بهذا الشكل:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```

2. **تشغيل الحاويات (Containers):**
   افتح موجه الأوامر (Terminal) في المجلد الجذري للمشروع وقم بتشغيل الأمر التالي:
   ```bash
   docker-compose up --build
   ```

3. **استخدام التطبيق:**
   - ستعمل الواجهة الأمامية على الرابط: `http://localhost:3000`
   - سيعمل السيرفر الخلفي على الرابط: `http://localhost:8000`
   
   قم بفتح متصفحك على `localhost:3000`، ارفع مستنداً، وابدأ بالاستكشاف!
